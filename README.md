A tool for managing system environments.

## Development setup

* install [Go](https://golang.org/)
* set your `$GOPATH`
* clone the repository into `{$GOPATH}/src/github.com/appPlant/alpinepass`
* `cd` into the repository
* run `go get -v`
* run alpinepass with `go run alpinepass.go`
